Installation

To install mod_time, go to the third-party module area in the proftpd source code and unpack the mod_time source tarball: 
  $ cd proftpd-dir/contrib/
  $ tar zxvf /path/to/mod_time-version.tar.gz
after unpacking the latest proftpd-1.3.x source code. For including mod_time as a staticly linked module: 
  $ ./configure --with-modules=mod_time:...
To build mod_time as a DSO module: 
  $ ./configure --enable-dso --with-shared=mod_time:...
Then follow the usual steps: 
  $ make
  $ make install
